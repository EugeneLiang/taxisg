Singapore taxis data viewer developed by U-Zyn Chua.

Data is obtained via deobfuscated API from a mobile app and may not represent actual data from Land Transport Authority of Singapore.

 

Data is collected and stored on a personal AWS account. Please be gentle when use.
If you wish to obtain the full data set, please contact me instead of brute forcing the database.
